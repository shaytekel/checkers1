# Checkers

P2P game using socket.io

---
## Requirements

- nodeJs
- javaScript version: ES6
- redis

---

## Install

    $ cd chekers
    $ yarn install

## Configure app

Open `chekers/config.ts` then edit it with your settings. 

- server port;
- redis server port;

## Running

open source code in your browser in order to see the state of game.
